# 🎮 Multi-Language Tic-Tac-Toe

A cross-platform Tic-Tac-Toe project featuring three distinct versions.

## 🚀 Project Versions

- Web App: HTML/CSS/JS with a visual menu.
- Python Edition: Terminal-based game with a "Random AI" mode.
- C++ Version: High-performance console implementation.

## 🛠️ Setup

- Web: Open front/welcome.html in any browser.
- Python: Run py game.py (Standard library only).
- C++: Compile game.cpp using any C++ compiler.

## 📁 Structure

- /front: All web-related files.
- /back: Python and C++ source code.
